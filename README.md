# 7VNR

Game RPG Offline được phát triển bằng Unity và C#.

## Giới thiệu

7VNR là project game RPG lấy cảm hứng từ phong cách game nhập vai cổ điển.  
Project tập trung phát triển gameplay chiến đấu, hệ thống nhân vật, quái vật và các chức năng cơ bản của một game RPG.

## Tính năng hiện có

- Di chuyển nhân vật
- Hệ thống đánh thường và skill
- Animation chiến đấu
- AI quái vật
- Chọn mục tiêu enemy
- Inventory
- UI gameplay
- Quản lý scene

## Công nghệ sử dụng

- Unity Engine
- C#
- Unity Animator
- Unity UI
- Spine 2D
- Git & GitHub
